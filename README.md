# Himanshu Mishra — Portfolio Website

A personal portfolio website built to present my skills, projects, certifications, participation, education, and contact information in a clean, modern interface.

The site uses a dark terminal-inspired design and is built with plain **HTML**, **CSS**, and **JavaScript**, keeping it lightweight, responsive, and easy to maintain.

## Features

- Responsive single-page layout
- Dark terminal-style visual design
- Smooth navigation and scroll interactions
- Sections for skills, projects, participation, SAP credentials, certificates, education, activities, and contact details
- Scroll reveal animations and subtle card hover effects
- Mobile-friendly navigation menu
- Downloadable/viewable certificate and credential assets

## Tech Stack

| Technology | Purpose |
| --- | --- |
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, responsiveness, and animations |
| JavaScript | Navigation behavior, scroll effects, and interactions |

## Sections Included

- **About** — introduction and profile summary
- **Skills** — languages, tools, platforms, and concepts
- **Projects** — selected personal and technical work
- **Participation** — hackathons and events
- **SAP** — SAP certification and badge details
- **Certificates** — cloud and course credentials
- **Education & Activities** — academic background and involvement
- **Contact** — email and social links

## Running the Project Locally

Because this is a static website, no build step is required.

1. Clone or download the repository.
2. Open `index.html` directly in a browser.

For a simple local server, you can also run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
